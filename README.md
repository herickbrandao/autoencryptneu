# autoencryptneu
Simple one-file text editor for sensitive data. Built in Neutralinojs

## neutralino
```
# install
npm install -g @neutralinojs/neu

# run
neu run

# build
neu build
```

## how to use
CTRL+S - Save file

CTRL+T - Change password

CTRL+R - Reload

For reset it all, delete all "data" file content, then reload your application
